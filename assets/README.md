# NelShop Assets

Organize project assets in this folder.

## Structure

| Folder | Purpose |
|--------|---------|
| `css/` | Stylesheets (styles.css) |
| `js/` | JavaScript (store.js, app.js, category.js, checkout.js, product-details.js) |
| `images/` | Product images, hero banners, logos |
| `icons/` | Favicon, UI icons, social icons |
| `fonts/` | Custom web fonts (if not using Google Fonts) |

## Usage

- **Product images:** Add local product photos here and update `store.js` to use paths like `assets/images/products/earbuds.jpg`
- **Favicon:** Place `favicon.ico` in `icons/` and add `<link rel="icon" href="assets/icons/favicon.ico">` to HTML
- **Logo:** Add `logo.svg` or `logo.png` in `images/` for the NelShop header
