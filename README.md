# NelShop

Premium e-commerce platform for **Asaba, Delta State**. Same-day delivery to Asaba, Okpanam, Koka & surrounding areas.

## Tech Stack

- **HTML5** – Semantic, SEO-friendly markup
- **Tailwind CSS** – Via Play CDN (prototyping); PostCSS build recommended for production
- **Vanilla JavaScript** – Cart logic, mobile menu, UI transitions

## Quick Start

1. Open `index.html` in a browser, or
2. Run a local server:
   ```bash
   npx serve .
   # or
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000` (or the port shown)

## Features

- **Dynamic Header** – Search bar, location badge, sticky glassmorphism nav
- **Hero Section** – Flash sales & fastest delivery messaging
- **Category Grid** – Electronics, Fashion, Groceries, Local Specialties
- **Product Cards** – Hover effects, add-to-cart animation, ₦ pricing, stock indicators
- **Trust Signals** – Same-day delivery, Pay on Delivery badges
- **Cart Drawer** – Slide-out cart with real-time totals
- **Mobile-First** – Responsive layout with mobile menu

## Local SEO

Metadata and copy include: Asaba, Okpanam, Koka, Ogbogonogo, Nnebisi Road, Summit Road.
